# run the code agent

```bash
python code-agent/terminal_toolkit_read_file.py
```

it will read the task list from the `task_list.json` file and run the code agent for each task.
The generated script are saved in the `task-script` directory.
The logs of tool calls and chat history are saved in the `logs` directory.


- the previous generated script are saved in the `task-script` directory.

- the logs of tool calls and chat history are saved in the `logs_read_file` directory.
